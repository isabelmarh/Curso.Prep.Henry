1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

* `for`
Crea un bucle o loop que se utiliza para repetir una o más instrucciones.
Por ejemplo,  
for (let paso = 0; paso < 5; paso++) {
  // Se ejecuta 5 veces, con valores del paso 0 al 4.
  console.log('Camina un paso hacia adelante');
}
* `&&`
El operador && se entiende como si fuera un "y". 
Por ejemplo let a = true && true // devuelve true

`||`, 
El operador `||` se entiende como si fuera un "o". 
let b =  true || false; // devuelve true

`!`
El operador `!` se entiende como si fuera un "no" logico o lo opuesto 
let c = !true;  // devuelve false
