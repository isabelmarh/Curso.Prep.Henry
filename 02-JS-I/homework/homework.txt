	* Variables
    En Javascript las variables son como una "caja" para guardar datos.  A esta caja le podemos poner un nombre anteponiendo "let".  Por ejemplo let saludo = "Hola".  También le ponemos anteponer const a valores que no van a cambiar.  Por ejemplo const miCumpleaños = '01.01.2001'

	* Strings
    En Javascript un string es una serie de caracteres que están entre comillas que pueden ser simples o dobles.  Por ejemplo let string = "Kia Morning"

	* Funciones (argumentos, `return`)
    Las funciones son una serie de pasos que calculan un valor o un procedimiento.  Una funcion consiste en usar la palabra function seguido por el nombre de una funcion.  Luego esta funcion puede aceptar uno o varios inputs o argumentos y retorna un output o resultado.  Por ejemplo funcionSuma(a, b) { return a + b}.

	* Declaraciones `if`

    function sonIguales(x, y) {
    if (x === y) { return true}
    return false
