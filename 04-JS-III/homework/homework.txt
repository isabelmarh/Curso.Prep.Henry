	* Arrays

Los arrays, arreglos o matrices son una lista de objectos y es una manera ordenada de juntar objetos bajo el nombre de un variable.  Por ejemplo,

let carritoDeCompras = ['huevos', 'queso', 'yogurt', 'frambuesas', 'frutillas'];

También puedes almacenar otros elementos como por ejemplo, cadena, número, objeto, otra variable, incluso otra matriz.
let número = [1, 1, 2, 3, 5, 8, 13];

Puedes acceder a elementos individuales en la matriz usando corchetes.
let carritoDeCompras[0] // devuelve 'huevos'

Hay varias propiedades que tiene un arreglo, por ejemplo podemos saber cuántos elementos hay dentro usando length.
numero.length; // devuelve 7
